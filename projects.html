2D Image Printer Robot

A robotic drawing system that converts a digital image into a physical drawing on a cylindrical surface. MATLAB is used for image processing, while Arduino-based motor control coordinates stepper motors and servos to synchronize cylinder rotation, pen translation, and pen contact.

Robotic Arm Application (AR + Voice Control)

A voice-controlled robotic arm interface integrating Augmented Reality (Unity) with ROS. Voice commands are translated into ROS actions and executed in a Gazebo simulation using MoveIt! for motion planning and evaluation.

Malaria Cell Image Classification (CNN + Streamlit)

An end-to-end deep learning project that classifies blood smear cell images as Uninfected or Parasitized using a TensorFlow CNN. Includes a professional Streamlit web application for upload-and-predict inference, confidence visualization, and a reproducible training and CLI pipeline.

Smart Waiting Rooms (IoT Occupancy Monitoring)

An IoT system for monitoring seat occupancy in hospital waiting rooms using ESP8266 nodes, ESP-NOW wireless communication, MQTT, and a Node-RED dashboard, demonstrating an embedded-to-cloud pipeline.

ESP32 Cyclic Executive Monitor (Embedded Real-Time Systems)

An embedded systems project implementing a cyclic-executive–style scheduler on an ESP32 to execute multiple periodic real-time tasks. The system performs deterministic waveform generation, frequency measurement on two digital inputs, analog signal monitoring with moving-average filtering, and periodic serial logging — all without using an RTOS.

Mars Rover Navigation (Behaviour-Based vs Evolutionary Robotics)

A comparative robotics project analysing Behaviour-Based Robotics (BBR) and Evolutionary Robotics (ER) approaches for autonomous Mars rover navigation in a Webots simulation environment. The rover follows beacon-assisted routes, performs line following, avoids obstacles, and reaches a reward zone, with performance evaluated in terms of speed, robustness, and adaptability.
